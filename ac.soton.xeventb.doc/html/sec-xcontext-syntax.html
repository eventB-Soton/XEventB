5.2 XContext Syntax

XCONTEXT ::=

   context ID

   [extends ID { ID }]

   [sets XSET {XSET}]

   [constants XCONSTANT { XCONSTANT }]

   [axioms XAXIOM {XAXIOM}]

   end

XSET ::= ID

XCONSTANT ::= ID

XAXIOM ::= [theorem] XLABEL "XPREDICATE"
